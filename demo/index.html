tabbed-interface Demo

The <tabbed-interface> web component transforms heading-structured content into an accessible tabbed interface.

Basic Example

Getting Started

This is the first tab panel content. It demonstrates the basic usage of the tabbed interface component.

Simply place your content with consistent heading levels, and the component will automatically create tabs.

Features

  • Automatic tab creation from headings
  • Full keyboard navigation support
  • ARIA attributes for accessibility
  • CSS custom properties for styling
  • Hash-based navigation

Installation

npm install @aarongustafson/tabbed-interface

Then import in your JavaScript:

import '@aarongustafson/tabbed-interface';
